# 🌐 沖縄県高齢者協同組合 Webサイト (Astro版)

Google Sitesから移行された、沖縄県高齢者協同組合の公式サイトです。
表示速度の向上と管理コストの削減を目的に、Astroを使用して構築されています。

## 🚀 技術スタック
- **フレームワーク**: Astro
- **コンテンツ管理**: Markdown (Content Collections)
- **ホスティング**: GitHub Pages (予定)

## 📁 プロジェクト構造
- `src/pages/`: 各固定ページ（ホーム、事業案内など）
- `src/content/news/`: お知らせ（ニュース）の本文（Markdown形式）
- `public/images/`: サイトで使用する画像資産

## 🧞 コマンド
プロジェクトのルートディレクトリで実行してください：

| コマンド | 内容 |
| :--- | :--- |
| `npm run dev` | ローカル開発サーバーを起動 (`localhost:4321`) |
| `npm run build` | 本番用の静的ファイルをビルド (`./dist/`) |
| `npm run preview` | ビルドした内容をローカルでプレビュー |

## 📝 更新方法
- **お知らせの追加**: `src/content/news/` に新しい `.md` ファイルを作成してください。
- **固定ページの編集**: `src/pages/` 内の該当する `.astro` ファイルを編集してください。
